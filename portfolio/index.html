Projects

SAI42 – Smart Automated Irrigation

SAI42 is an intelligent irrigation system that integrates ESP32-based hardware with a web platform. It allows users to monitor their plants’ health by tracking soil moisture, temperature, humidity, lighting, and weather conditions. The system automatically waters plants based on soil moisture levels and offers remote control through a real-time dashboard with manual override options. Designed to simplify smart gardening, SAI42 helps make plant care more efficient and convenient.

Keypad Door Lock

A basic Arduino-powered door lock system that uses matrix keypad for password entry. It features the ability to change and store the password in EEPROM, ensuring the password persists even after a power lost, making it ideal for simple yet effective locks.

ESP8266 YouTube Subscriber Counter

An IoT project that leverages the ESP8266 microcontroller to fetch and display real-time YouTube subscriber counts using the YouTube Data API. Designed for creators and tech enthusiasts, it blends cloud API integration with physical device output for a live, connected experience.
